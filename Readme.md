# Rolling Scopes School NodeJS 2025Q4

This repository contains several project branches for the NodeJS 2025Q4 curriculum. The following is a brief overview of projects found across branches.

- [basic-testing](https://github.com/Tubyliec/rolling-scopes-school-NodeJS-2025Q4/tree/basic-testing) — Basic testing exercises and examples (unit/integration tests).
- [battleship](https://github.com/Tubyliec/rolling-scopes-school-NodeJS-2025Q4/tree/battleship) — Battleship game implementation (game logic and UI/CLI).
- [crud-api](https://github.com/Tubyliec/rolling-scopes-school-NodeJS-2025Q4/tree/crud-api) — RESTful CRUD API project demonstrating create/read/update/delete endpoints.
- [file-manager](https://github.com/Tubyliec/rolling-scopes-school-NodeJS-2025Q4/tree/file-manager) — File manager application (file operations, CLI or HTTP interface).
- [nodejs-basics](https://github.com/Tubyliec/rolling-scopes-school-NodeJS-2025Q4/tree/nodejs-basics) — Node.js fundamentals and small exercises.
- [task-graphql](https://github.com/Tubyliec/rolling-scopes-school-NodeJS-2025Q4/tree/task-graphql) — GraphQL API task and schema/resolvers examples.
- [nodejs2025Q4-service-part-1](https://github.com/Tubyliec/rolling-scopes-school-NodeJS-2025Q4/tree/nodejs2025Q4-service-part-1) — Service — Part 1 (core backend service tasks).
- [nodejs2025Q4-service-part-2](https://github.com/Tubyliec/rolling-scopes-school-NodeJS-2025Q4/tree/nodejs2025Q4-service-part-2) — Service — Part 2 (additional features and integrations).
- [nodejs2025Q4-service-part-3](https://github.com/Tubyliec/rolling-scopes-school-NodeJS-2025Q4/tree/nodejs2025Q4-service-part-3) — Service — Part 3 (finalization, tests, or deployment).
